# 注音符號規則筆記：聲符、介符、韻符、聲調

台灣國語注音符號（Zhuyin Fuhao / Bopomofo）本身的拼寫規則，跟 [zhuyin-research.md](zhuyin-research.md)（技術/library 研究）是不同主題：這篇整理的是注音符號**語言學上的組成規則**，來源主要是教育部《國語注音符號手冊》。

## 總覽：37 個符號 = 聲符 21 + 介符 3 + 韻符 13

歷史上還有 3 個聲符（ㄪㄫㄬ）跟 1 個韻符（ㄭ）已停用：

> 聲符「ㄪ」、「ㄫ」「ㄬ」國音不用。韻符「ㄭ」注音時省略不標。

`ㄭ`（空韻）雖然不寫出來，但概念上還在——這就是為什麼 `ㄓㄔㄕㄖㄗㄘㄙ` 可以自己單獨當一個完整音節（見下方「整體認讀」）。

## 聲符（21 個）— 依發音部位分組

| 分類                 | 符號       |
| -------------------- | ---------- |
| 雙唇音                | ㄅ ㄆ ㄇ    |
| 唇齒音                | ㄈ         |
| 舌尖音                | ㄉ ㄊ ㄋ ㄌ |
| 舌根音                | ㄍ ㄎ ㄏ    |
| 舌面音                | ㄐ ㄑ ㄒ    |
| 捲舌音（翹舌音）       | ㄓ ㄔ ㄕ ㄖ |
| 舌尖前音（平舌音）     | ㄗ ㄘ ㄙ    |

## 介符（3 個）— ㄧ ㄨ ㄩ

介符夾在聲符跟韻符中間，決定「四呼」分類：

| 四呼   | 條件                     |
| ------ | ------------------------ |
| 開口呼 | 沒有介符，主要元音不是 i/u/ü |
| 齊齒呼 | 介符或主要元音是 ㄧ (i)   |
| 合口呼 | 介符或主要元音是 ㄨ (u)   |
| 撮口呼 | 介符或主要元音是 ㄩ (ü)   |

四呼決定了哪些聲符可以搭配哪個介符（見下方相容規則）。

## 韻符（13 個）

`ㄚ ㄛ ㄜ ㄝ ㄞ ㄟ ㄠ ㄡ ㄢ ㄣ ㄤ ㄥ ㄦ`

其中 `ㄦ`（er）比較特殊：可以自己獨立成一個音節（兒、耳），也可以當「兒化韻」黏在別的韻符後面（一點兒）。

## 音節組成規則

一個完整音節的結構：

```
[聲符]（可省略） + [介符]（可省略） + 韻符（通常必要） + 聲調
```

例外：`ㄓ ㄔ ㄕ ㄖ ㄗ ㄘ ㄙ` 這 7 個聲符可以**自己單獨成一個音節**，不需要接韻符——因為它們後面接的是那個停用、不寫出來的空韻 `ㄭ`。例：

- 資 zī → `ㄗ`（不是 `ㄗㄭ`）
- 是 shì → `ㄕˋ`

這也是我們專案 [zhuyin.ts](zhuyin.ts) 裡 `toZhuyinSegments()` 要處理的一種常見情況：`symbols` 陣列可能只有一個聲符字元，沒有額外韻符。

### 介符與聲符的相容規則（哪些組合國語裡不存在）

- `ㄐ ㄑ ㄒ`（舌面音）**只能**接 `ㄧ` 或 `ㄩ`，絕不會接 `ㄨ`。
- `ㄓ ㄔ ㄕ ㄖ ㄗ ㄘ ㄙ`（捲舌音、平舌音）**絕不會**接 `ㄧ` 或 `ㄩ`，只能接 `ㄨ` 或不接介符。
- `ㄅ ㄆ ㄇ ㄈ`（唇音）只能接 `ㄨ` 或 `ㄧ`，不接 `ㄩ`；且能接 `ㄨ` 的組合很少（`ㄅㄨ`、`ㄆㄨ`、`ㄇㄨ`、`ㄈㄨ`）。

手冊裡把這類組合限制整理成表格，空格處就是「國音缺音」——不是遺漏，是這個組合在標準國語裡本來就不存在。

### 零聲母（沒有聲符的音節）

介符 `ㄧ ㄨ ㄩ` 自己也可以是整個音節的開頭（沒有聲符），例：有 yǒu → `ㄧㄡˇ`。這點注音符號跟漢語拼音不一樣——拼音會把這種零聲母的 i/u/ü 改寫成 y-/w-（you, wu），注音符號則不需要，直接寫 `ㄧㄡˇ`、`ㄨˇ`，維持原樣。

## 聲調（四聲 + 輕聲）

| 調類          | 符號 | 說明                         |
| ------------- | ---- | ---------------------------- |
| 第一聲（陰平） | 無   | 高平調，習慣上**不標符號**    |
| 第二聲（陽平） | `ˊ`  | 上揚調                        |
| 第三聲（上聲） | `ˇ`  | 先降後升                      |
| 第四聲（去聲） | `ˋ`  | 下降調                        |
| 輕聲           | `˙`  | 短而輕，位置規則跟前四聲不同  |

### 標寫位置規則（教育部原文）

> 陰、陽、上、去四聲調號，無論直式或橫式，標注在字音最後一個符號右上角。

> 輕聲調號標注，直式注音時標注在字音上方，橫式注音時標注在字音前方。

也就是說：

- **二、三、四聲**：不管橫式直式，都貼在整個音節**最後一個符號的右上角**。
- **輕聲**：直式（傳統書本直排）標在音節**正上方**；**橫式**（一般網頁橫排）標在音節**最前面**（左邊）。

這跟我們專案的實作有個值得記錄的落差：[ZhuyinText.vue](ZhuyinText.vue) 目前是「整句橫排、但每個字的注音符號自己直排疊成一欄」的混合排法（因為 CSS 原生沒有可靠的橫排注音右側方案，見 [zhuyin-research.md](zhuyin-research.md)）。輕聲點目前是疊在符號欄「左上角」，比較接近直式規則（標在音節上方，偏左因為我們是疊在單一欄位的角落），而不是嚴格的橫式規則（該整個貼到音節最前面、跟其他字同一水平線）。這是混合排法下的取捨，不是嚴格照教育部橫式規範來的，先記錄下來，之後如果要做到教科書等級的精確排版可以再調整。

`pinyin-to-zhuyin` 套件輸出的字串也印證了「輕聲在前」的橫式慣例：`p2z('bu5')` 回傳 `˙ㄅㄨ`，`˙` 就是排在最前面，不是接在後面。

## 變調（連讀時發音會變，但注音符號書寫通常不變）

以下屬於**語音層面**的變調規則，書寫上的注音符號一般維持本調不變（教材會標本調，實際唸法約定俗成），但列出來是因為會影響「這個字到底該顯示第幾聲」的判斷：

- **上聲變調**：兩個第三聲字相連，前字唸成第二聲（但符號仍標第三聲）。例：你好 nǐhǎo 唸起來像 níhǎo。
- **「不」字變調**：「不」單獨或在非第四聲字前唸第四聲 `ㄅㄨˋ`；在第四聲字前變第二聲，例：不是 búshì `ㄅㄨˊㄕˋ`；在「A不A」正反問句中間常唸輕聲，例：能不能 `néng‧bù néng`。
- **「一」字變調**：單念、位於詞尾、序數詞中唸第一聲（第一 dìyī）；在第四聲字前變第二聲（一定 yídìng）；在第一、二、三聲字前變第四聲（一天 yìtiān、一年 yìnián、一種 yìzhǒng）。

這幾條規則跟我們 [zhuyin.ts](zhuyin.ts) 目前用的 `pinyin-pro` 有關：pinyin-pro 是查字典/分詞猜讀音，「能不能」裡的「不」能正確判斷成輕聲，代表它的詞庫已經處理了一部分這類變調案例；但這不是通用的變調演算法，遇到字典沒收錄的組合時未必會套用「不」「一」變調規則。這也是為什麼之前的研究筆記會建議「多音字/變調需要人工抽查」。
